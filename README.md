# 🛒 E-Commerce Backend API

A complete E-Commerce Backend API built with Django REST Framework.

## 🚀 Features

- JWT Authentication
- Categories CRUD
- Products CRUD
- Cart
- Wishlist
- Reviews
- Orders
- Search
- Filtering
- Ordering
- Pagination
- Swagger API Documentation

## 🛠️ Tech Stack

- Python
- Django
- Django REST Framework
- PostgreSQL
- Simple JWT
- drf-spectacular (Swagger)

## 📖 API Documentation

- Swagger: `/api/docs/`
- ReDoc: `/api/redoc/`

## 🔑 Authentication

JWT Authentication using Access and Refresh Tokens.
